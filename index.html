Overview

Purpose

This website will be a display of my favorite K-pop songs, artists, and more. It will include basically anything related my preferences of K-pop stuff.

Target Audience

People that enjoy K-pop and are looking for song recommendations, new music, or a different perspective on K-pop stuff.

Planned Website Pages

  • Home Page: A description of the website and why it exists
  • Song Recommendations: Songs I would recommend based on mood, experience, etc.
  • About Me: Bio, my experience with K-pop, my favorite kpop groups, link to my playlists
  • Contact: Contact me

Visual Design Choices

Colors

The website will use a very simple color palette of blue, black, and white, because they're pretty high-contrast colors, and I want to keep the website as simple as possible to put more focus on the actual content.

Font

The website will only use Verdana to increase cohesiveness and because it's highly legible.

Layout

The layout will be similar to what it looks like right now for most of the pages. The Song Recommendations page will structure each song in a separate box using CSS.
